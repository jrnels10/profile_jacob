About me

Personal

Seeing new environments, experiencing new cultures and taking on new challenges is what drives me to explore the world we live in. This had led me to pursue a degree in Geography and an interest in GIS.

Tech

My specialty is working with mapping applications such as ESRI ArcGIS API using JavaScript as a main programming langauge. Geography/GIS
